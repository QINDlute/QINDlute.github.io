HTML基础 ​

什么是HTML ​

HTML（HyperText Markup Language）是用于创建网页的标准标记语言。它描述了网页的结构，通过一系列元素（elements）来定义内容的含义和外观。

基本结构 ​

html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
</head>
<body>
    <h1>这是一级标题</h1>
    <p>这是一个段落。</p>
</body>
</html>

常用标签 ​

文本标签 ​

  • <h1> - <h6>：标题标签，h1最大，h6最小
  • <p>：段落标签
  • <strong>：粗体文本
  • <em>：斜体文本
  • <a>：链接标签，用于创建超链接

列表标签 ​

  • <ul>：无序列表
  • <ol>：有序列表
  • <li>：列表项

媒体标签 ​

  • <img>：图片标签
  • <audio>：音频标签
  • <video>：视频标签

最佳实践 ​

  1. 语义化HTML：使用合适的标签描述内容，提高可访问性和SEO
  2. 合理的嵌套：确保标签嵌套正确，避免交叉嵌套
  3. 使用注释：为复杂代码添加注释，提高可维护性
  4. 响应式设计：使用meta viewport标签和响应式布局技术
  5. 性能优化：压缩HTML代码，优化图片和资源加载

学习资源 ​

  • MDN Web Docs - HTML
  • W3Schools - HTML
  • HTML Living Standard

总结 ​

HTML是Web开发的基础，掌握HTML的基本概念和常用标签是成为前端开发者的第一步。通过不断实践和学习，可以逐渐掌握更高级的HTML特性和最佳实践。
