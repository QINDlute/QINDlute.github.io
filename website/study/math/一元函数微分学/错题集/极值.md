# 错题集-极值

判断极值的方法：[极值判别法](/study/math/一元函数微分学/2-极值.html#极值判别法)

::: faq-math 例1.\;f'(1)=0.\lim_{x \to 1} {\frac{f(x)}{\sin {\pi x}}} = -2,问x=1是极大点还是极小点?#
解:  
<Badge type="tip">法一:</Badge>

$\exists \delta > 0, 当\; 0 < |x-1| < \delta \;时.\frac{f'(x)}{\sin {\pi x}} < 0.$  
$\left\{\begin{matrix} 
  f'(x)<0, x \in (1-\delta,1) \\  
  f'(x)>0, x \in (1,1+\delta)
\end{matrix}\right\}\Rightarrow x=1为极小点.$(极限保号性)  
<Badge type="tip">法二:</Badge>

$f'(1)=0.$  
$-2 = \lim_{x \to 1} {\frac{f'(x)}{\sin {[\pi + \pi(x-1)}}} = -\lim_{x \to 1} {\frac{f'(x)}{\pi(x-1)}} = -\frac{1}{\pi} \lim_{x \to 1} \frac{f'(x)-f'(1)}{x-1} = -\frac{1}{\pi}f''(1)$  
$\Rightarrow f''(1) = 2\pi > 0$  
$\therefore x=1为极小点.$

:::
