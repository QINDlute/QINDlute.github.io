# 不定积分-工具
(一) 基本公式  
1. 常数  
$\int k\mathrm{d}x = kx + C;$

2. 幂函数不定积分公式  
① $a \ne -1:\int x^a\mathrm{d}x = \frac{1}{a+1}x^{a+1} + C;$  
② $a=-1:\int x^{-1}\mathrm{d}x = \ln |x| + C;$
::: details 简单证明
$\because x < 0 时,[\ln (-x)]' = \frac{1}{x},\therefore \int \frac{1}{x}\mathrm{d}x = \ln (-x) + C.$  
$x > 0 时,(\ln x)' = \frac{1}{x},\therefore \int \frac{1}{x}\mathrm{d}x = \ln x + C.$  
$\therefore \int \frac{1}{x}\mathrm{d}x = \ln |x| + C.$3
:::

3. 指数函数  
① $a \ne e:\int a^x\mathrm{d}x=\frac{a^x}{\ln a} + C;$  
② $a = e:\int e^x\mathrm{d}x = e^x + C;$

4. 三角函数  
① $\int \sin x\mathrm{d}x = -\cos x + C;$  
② $\int \cos x\mathrm{d}x = \sin x + C;$  
③ $\int \tan x\mathrm{d}x = \ln |\sec x| + C;$  
④ $\int \cot x\mathrm{d}x = \ln |\sin x| + C;$  
⑤ ⭐$\int \sec x\mathrm{d}x = \ln |\sec x + \tan x| + C;$  
⑥ ⭐$\int \csc x\mathrm{d}x = \ln |\csc x - \cot x| + C;$  
⑦ $\int \sec^2 x\mathrm{d}x = \tan x + C;$  
⑧ $\int \csc^2 x\mathrm{d}x = - \cot x + C;$  
⑨ $\int \sec x \tan x\mathrm{d}x = \sec x + C;$  
⑩ $\int \csc x \cot x\mathrm{d}x = - \csc x + C;$

::: details 推导
$\int \tan x\mathrm{d}x = \int \frac{\sin x}{\cos x}\mathrm{d}x = \int - \frac{\mathrm{d}\cos x}{\cos x}$  
$左 = \int \mathrm{d}(- \ln |\cos x|) = \int \mathrm{d}(\ln |\sec x|) = \ln |\sec x| + C.$
<br>

$\int \sec x\mathrm{d}x = \int \frac{1}{\cos x}\mathrm{d}x = \int \frac{\cos x}{\cos^2 x}\mathrm{d}x = \int \frac{\mathrm{d} \sin x}{\cos^2 x} = \int \frac{\mathrm{d} \sin x}{1 - \sin^2 x}$  
$令~t = \sin x,左 = \int \frac{1}{1 - t^2} \mathrm{d}t = \frac{1}{2}\int (\frac{1}{1 + t} + \frac{1}{1 - t}) \mathrm{d}t = \frac{1}{2}(\int \frac{1}{1 + t} \mathrm{d}(1+t) - \int \frac{1}{1 - t} \mathrm{d}(1-t))$  
$左 = \frac{1}{2}(\ln |1 + t| - \ln |1 - t|) + C = \frac{1}{2}\ln \frac{|1+t|}{|1-t|} + C = \frac{1}{2}\ln |\frac{1+\sin x}{1-\sin x}| + C$  
$左 = \frac{1}{2}\ln |\frac{(1+\sin x)^2}{1-\sin^2 x}| + C = \ln |\frac{1+\sin x}{\cos x}| + C = \ln |\sec x + \tan x| + C$
:::

5. 有理分式  
① $\int \frac{\mathrm{d}x}{\sqrt{1-x^2}}=\arcsin x + C;$  
② $\int \frac{\mathrm{d}x}{\sqrt{a^2-x^2}}=\int \frac{\mathrm{d}(\frac{x}{a})}{\sqrt{1-(\frac{x}{a})^2}} = \arccos \frac{x}{a} + C;$  
③ $\int \frac{\mathrm{d}x}{1+x^2} = \arctan x + C;$  
④ $\int \frac{\mathrm{d}x}{a^2+x^2} = \frac{1}{a}\int \frac{\mathrm{d}(\frac{x}{a})}{1+(\frac{x}{a})^2} = \frac{1}{a}\arctan \frac{x}{a} + C;$  
⑤ $\int \frac{\mathrm{d}x}{x^2-a^2} =\frac{1}{2a}[\int \frac{\mathrm{d}(x-a)}{x-a} - \int \frac{\mathrm{d}(x+a)}{x+a}] = \frac{1}{2a} \ln |\frac{x-a}{x+a}| + C;$
