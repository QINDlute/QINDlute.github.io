# 指数函数导数证明

$对于\;a^x,证明其导数(a^x)'=a^x\ln a.$

::: whiteboard 自然常数 e
## 前提① 定义自然对数的底 e
数学上通常将 e 定义为一个极限:  
$e = \lim_{n \to \infty} (1+ \frac{1}{n})^n$  
或者等价地，在连续变量下:  
$\lim_{t \to 0} (1+ t)^{\frac{1}{t}} = e$
:::


## 方法一:导数定义 ①

$f(x)=a^x$  
$f'(x) = \lim_{\Delta x \to 0} \frac{f(x+\Delta x)-f(x)}{\Delta x} = \lim_{\Delta x \to 0} \frac{a^{x+\Delta x}-a^x}{\Delta x} = a^x\lim_{\Delta x \to 0} \frac{a^{\Delta x}-1}{\Delta x}.$  
$设\;L = \lim_{h \to 0} \frac{a^h-1}{h}$  
$令\;a^h-1 = t \Rightarrow h = \log_a (1 + t),当\;h \to 0 \Rightarrow t \to 0.$  
$L = \lim_{t \to 0} \frac{t}{\log_a (1 + t)} = \lim_{t \to 0} \frac{1}{\frac{1}{t}\log_a (1 + t)} = \lim_{t \to 0} \frac{1}{\log_a (1 + t)^{\frac{1}{t}}}$  
$\because \lim_{t \to 0} (1 + t)^{\frac{1}{t}} = e,$ 且已知**对数函数是连续的**(可导必然连续)  
$\therefore L = \frac{1}{\ln_a e}=\ln a$  
$\therefore f'(x) = a^x\lim_{\Delta x \to 0} \frac{a^{\Delta x}-1}{\Delta x} = a^x\cdot L = a^x\ln a.$

::: warning 关于极限'路上'与'终点'的悖论
在数学上,函数 $f(x)$ 具有连续性的定义之一就是极限符号可以与函数符号互换位置.  
即: $\lim_{x \to x_0} f(g(x)) = f(\lim_{x \to x_0} g(x))$  
如 $\lim_{t \to 0}\log_a (1 + t)^{\frac{1}{t}} = \log_a \lim_{t \to 0}(1 + t)^{\frac{1}{t}}$, 如果 $\log_a x$ 不连续则该等式不成立.  
$设函数h(x):$  
$\left\{\begin{matrix} 
  h(x) = 100, x = e \\  
  h(x) = 0, x \ne e
\end{matrix}\right.$  
$则\;\left\{\begin{matrix} 
  \lim_{t \to 0}\log_a (1 + t)^{\frac{1}{t}} = 0; \\  
  \log_a \lim_{t \to 0}(1 + t)^{\frac{1}{t}} = 100.
\end{matrix}\right.$  
$\lim_{t \to 0} h(g(t))\;是动态趋近的过程,该过程先进行\;h(g(t))\;的运算,再整体取极限.$  
$在\;t \to 0\;的动态过程中,对于\;g(t),这意味着\;g(t)\;只是无限接近\;e\;但不等于 e.$  
$所以h(g(t))\equiv 0.$  
$而对于\;h(\lim_{t \to 0}g(x)),优先完成\;\lim_{t \to 0} g(t)\;的极限运算(直接提取静态目标)$  
$得到的结果是\;h(e)=100.$

:::


::: whiteboard 对数函数求导
## 前提② 证明对数函数的导数
<Badge type="note">证:</Badge>

$f(x)=\ln x$  
$根据导数定义:f'(x) = \lim_{\Delta x \to 0} \frac{\ln(x+\Delta x) - \ln x}{\Delta x}.$  
$左 = \lim_{\Delta x \to 0} \frac{\ln(1 + \frac{\Delta x}{x})}{\Delta x} = \lim_{\Delta x \to 0} \ln (1 + \frac{\Delta x}{x})^{\frac{1}{\Delta x}} = \lim_{\Delta x \to 0} \ln [(1 + \frac{\Delta x}{x})^{\frac{x}{\Delta x}}]^{\frac{1}{x}}.$  
$\because 当\;\Delta x \to 0 时,\frac{\Delta x}{x} \to 0,\frac{x}{\Delta x} \to \infty$  
$\therefore f'(x) = \ln e^{\frac{1}{x}} = \frac{1}{x}.$

:::


## 方法二:隐函数 ①②

$y=a^x$  
$两边取对数:\ln y = \ln a^x = x\ln a$  
$两边对 x 求导:左边是复合函数\;\ln(y(x)),导数为\;\frac{1}{y}y'$  
$\Rightarrow \frac{1}{y}y' = \ln a$  
$\Rightarrow y' = y\ln a$  
$\Rightarrow (a^x)' = a^x\ln a$


::: whiteboard 自然指数函数求导
## 前提③ 证明自然指数函数的导数
<Badge type="note">证:</Badge>

$\because y = e^x$ 是 $x = \ln y$ 的反函数  
根据反函数的求导法则:$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{1}{\frac{\mathrm{d}x}{\mathrm{d}y}}$ 且 $\frac{\mathrm{d}x}{\mathrm{d}y} = \frac{1}{y}$  
$\Rightarrow \frac{\mathrm{d}y}{\mathrm{d}x} = \frac{1}{\frac{1}{y}}=y=e^x$  
$即\;(e^x)'= e^x.$

:::


## 方法三:自然对数 ①②③

$y = a^x = e^{x\ln a}$  
$根据复合函数求导法则:\frac{\mathrm{d}}{\mathrm{d}x}(a^x) = e^{x\ln a}\cdot ln a$  
$\because e^{x\ln a} = a^x$  
$\therefore (a^x)' = a^x\ln a$
