# 错题集-极值

判断极值的方法：[极值判别法](/study/math/3_一元函数微分学/2-极值.html#极值判别法)

::: faq-math 例1.\;f'(1)=0.\lim_{x \to 1} {\frac{f(x)}{\sin {\pi x}}} = -2,问x=1是极大点还是极小点?#
解:  
<Badge type="tip">法一:</Badge>

$\exists \delta > 0, 当\; 0 < |x-1| < \delta \;时.\frac{f'(x)}{\sin {\pi x}} < 0.$  
$\left\{\begin{matrix} 
  f'(x)<0, x \in (1-\delta,1) \\  
  f'(x)>0, x \in (1,1+\delta)
\end{matrix}\right\}\Rightarrow x=1为极小点.$(极限保号性)
<Badge type="tip">法二:</Badge>

$f'(1)=0.$  
$-2 = \lim_{x \to 1} {\frac{f'(x)}{\sin {[\pi + \pi(x-1)]}}} = -\lim_{x \to 1} {\frac{f'(x)}{\pi(x-1)}} = -\frac{1}{\pi} \lim_{x \to 1} \frac{f'(x)-f'(1)}{x-1} = -\frac{1}{\pi}f''(1)$  
$\Rightarrow f''(1) = 2\pi > 0$  
$\therefore x=1为极小点.$

:::

::: faq-math 例2.\;0 < a < b.证:\ln {\frac{b}{a}} > \frac{2(b-a)}{a+b}.#
解:
<Badge type="tip">法一:</Badge>

$证:\ln {\frac{b}{a}} > \frac{2(b-a)}{a+b} \Leftrightarrow \ln {\frac{b}{a}} > \frac{2(\frac{b}{a}-1)}{1+\frac{b}{a}} \Leftrightarrow \ln {\frac{b}{a}} - \frac{2(\frac{b}{a}-1)}{1+\frac{b}{a}} > 0,\frac{b}{a} > 1.$  
$令\;f(x) = \ln x - \frac{2(x-1)}{1+x} ~(x > 1).$  
$f(1) = 0 - 0 = 0,f'(x) = \frac{1}{x} - \frac{4}{(1+x)^2} = \frac{(x-1)(x+3)}{x(1+x)^2} > 0 ~(x > 1).$  
$\left\{\begin{matrix} 
  f(1) = 0 \\  
  f'(x) > 0 ~(x > 1)
\end{matrix}\right\}\Rightarrow f(x) > 0 ~(x > 1).$  
$\therefore \ln x - \frac{2(x-1)}{1+x} > 0 ~(x > 1) \Rightarrow \ln {\frac{b}{a}} - \frac{2(\frac{b}{a}-1)}{1+\frac{b}{a}} > 0.$  
$\therefore \ln {\frac{b}{a}} > \frac{2(b-a)}{a+b}.$
<Badge type="tip">法二:</Badge>

$证:\ln {\frac{b}{a}} > \frac{2(b-a)}{a+b} \Leftrightarrow (a+b)(\ln b - \ln a) - 2(b-a) > 0.$  
$令 \varphi (x) = (a+x)(\ln x - \ln a) - 2(x-a) ~(x > a).\varphi (a)=0$  
$\varphi' (x) = \ln x - \ln a + \frac{a}{x} - 1,\varphi' (a) = 0;$  
$\varphi'' (x) = \frac{1}{x} - \frac{a}{x^2} = \frac{x-a}{x^2} > 0 ~(x > a).$  
$\left\{\begin{matrix} 
  \varphi' (a) = 0 \\  
  \varphi'' (x) > 0 ~(x > a)
\end{matrix}\right\}\Rightarrow \varphi' (x) > 0 ~(x > a).$  
$\left\{\begin{matrix} 
  \varphi (a) = 0 \\  
  \varphi' (x) > 0 ~(x > a)
\end{matrix}\right\}\Rightarrow \varphi (x) > 0 ~(x > a).$  
$\because b > a$  
$\therefore \varphi (b) > 0.$

:::
