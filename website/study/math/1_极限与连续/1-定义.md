# 极限与连续-定义
## 极限定义

$设函数~f(x)~在~x_0~的某个去心领域内有定义.$  
$若存在实数~A,有~\forall ~\varepsilon > 0,\exists ~\delta > 0,使得当~x~满足:$  
$0 < |x - x_0| < \delta$  
$时，恒有:$  
$|f(x) - A| < \varepsilon.$  
$则称函数~f(x)~在~x_0~处的极限值为~A,记作~\lim_{x \to x_0} f(x) = A.$

::: tip 极限定义解读
① 对于 $\forall ~\varepsilon > 0,\exists ~\delta > 0$,理解为:无论取的$\varepsilon$有多小,总能找到一个$\delta$满足条件.  
② $0 < |x - x_0| < \delta,|f(x) - A| < \varepsilon.$ 意为只要$x$距离$x_0$足够近,$f(x)$要多接近$A$就有多接近.  
③ $0 < |x - x_0| < \delta \Rightarrow x_0 - \delta < x < x_0 + \delta ~(x \ne x_0)$ 此时$x$在$x_0$的去心领域内.
:::


## 连续定义

$设函数~f(x)~在~x_0~的某个领域内有定义,且~\lim_{x \to x_0} f(x) = f(x_0).$  
$则称函数~f(x)~在~x_0~处连续.$

::: note 连续定义补充
函数连续是在函数极限的基础上定义的，也就是说函数连续必定极限存在.  
根据$\lim_{x \to x_0} f(x) = f(x_0)$可以得到两个信息:  
① $函数在~x_0~处有定义,即~f(x_0)~存在$  
② $极限~\lim_{x \to x_0}f(x)~存在$  

连续的完整定义:  
$设函数~f(x)~在~x_0~的某个领域内有定义.$  
$若存在实数~A,有~\forall ~\varepsilon > 0,\exists ~\delta > 0,使得当~x~满足:$  
$|x - x_0| < \delta$  
$时，恒有:$  
$|f(x) - f(x_0)| < \varepsilon.$  
$则称函数~f(x)~在~x_0~处连续,记作~\lim_{x \to x_0} f(x) = f(x_0).$
:::


下面一道例题使用$\varepsilon-\delta$语言的推导过程帮助你理解函数的极限与连续.

::: faq-math 例题. 设置函数~y=f[g(x)]~是复合函数.。
若~\lim_{x \to x_0} g(x) = A,且~f(u)~在点~u=A~处连续.。
证:\lim_{x \to x_0} f[g(x)] = f(A).#

<Badge type="note">证:</Badge>

$1^0.\because f(u)~在~A~点连续,根据连续定义:$  
$\forall ~\varepsilon > 0,\exists ~\eta > 0,使得当$  
$|u-A| < \eta,恒有:$  
$|f(u) - f(A)| < \varepsilon.$  
$2^0.\because \lim_{x \to x_0} g(x) = A,根据极限定义:$  
$对于~f~的~\eta > 0,\exists ~\delta > 0,使得当$  
$0 < |x - x_0| < \delta,恒有:$  
$|g(x) - A| < \eta.$  
$3^0.\because g(x)~落在~A~的~\eta~范围内,则根据\;f(u)\;在~A~点连续的定义推出:$  
$\forall ~\varepsilon > 0,\exists ~\eta > 0,使得当$  
$|g(x)-A| < \eta,恒有:$  
$|f(g(x)) - f(A)| < \varepsilon.$  
$4^0.结合~g(x)~在~x_0~处的极限定义,有:$  
$\forall ~\varepsilon > 0,\exists ~\delta > 0,使得当$  
$0 < |x - x_0| < \delta,恒有:$  
$|f(g(x)) - f(A)| < \varepsilon.$  
$按照定义,得:\lim_{x \to x_0} f[g(x)] = f(A).$

:::
