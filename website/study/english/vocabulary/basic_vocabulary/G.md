::: tip 格林规律{#green_law}
格林定律是由德国语言学家 Jacob Grimm 提出的音变规律，它描述了原始印欧语（PIE）中的辅音在进入日耳曼语族（包括英语、德语、荷兰语等）时，发生的系统性、有规律的清浊音变化。

| **PIE 辅音** | **日耳曼语对应** | **例子** |
| :---------: | :------------: | :------: |
| 清塞音：**p**, **t**, **k** | 清擦音: **f**, **θ**, **h** | pater (父亲) → 英语 **father** (p→f)；trei (三) → 英语 **three** (t→θ)；ḱard- (心) → 英语 **heart** (k→h) |
| 浊塞音：**b**, **d**, **g** | 清塞音: **p**, **t**, **k** | bʰreh₂tér (兄弟) → 英语 **brother** (b→p？不，这里是浊送气音，看第三组)；dōm- (房子) → 英语 **tomb** (d→t)；ǵnō- (知道) → 英语 **know** (g→k) |
| 浊送气塞音: **bʰ**, **dʰ**, **gʰ** | 浊塞音: **b**, **d**, **g** | bʰreh₂tér (兄弟) → 英语 **brother** (bʰ→b)；dʰeh₃- (做) → 英语 **do** (dʰ→d)；gʰrem- (咆哮) → 英语 **grim** (gʰ→g) |
:::
