## JavaScript 简介

JavaScript是一种轻量级的解释型编程语言，主要用于为网页添加交互功能。

## 变量与数据类型

JavaScript中的变量用于存储数据，支持多种数据类型，包括字符串、数字、布尔值、对象等。

## 运算符与表达式

JavaScript提供了多种运算符，用于执行算术、比较、逻辑等操作。

## 控制流程

JavaScript提供了条件语句和循环语句，用于控制代码的执行流程。

## 函数

函数是JavaScript中的基本构建块，用于封装可重用的代码块。
