## React 简介

React是一个用于构建用户界面的JavaScript库，由Facebook开发。

## JSX 语法

JSX是JavaScript的扩展语法，允许在JavaScript中编写类似HTML的代码。

## 组件与属性

React组件是构建React应用的基本单位，组件可以接收属性并渲染UI。

## 状态与生命周期

React组件的状态用于管理组件内部数据，生命周期方法用于控制组件的渲染过程。

## 事件处理

React提供了统一的事件处理机制，用于处理用户交互事件。
